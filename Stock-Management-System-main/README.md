# 📦 Stock Management System

A dynamic, fully responsive web application built to streamline inventory tracking, monitor dynamic stock levels, and simplify product data management. Developed as a part of the **4th Semester Full Stack Development (FSD) Lab** curriculum at SKIT.

🌐 **Live Demo:** https://stock-management-system-ebon.vercel.app

---

## 🚀 Key Features

- **Real-Time Inventory Tracking:** Efficiently monitor product additions, current stock quantities, and instant status updates.
- **Dynamic Dashboard:** A clean, intuitive, and interactive user interface designed for seamless navigation and data visualization.
- **Data Integrity & Validation:** Built-in checks to avoid inventory counting discrepancies and maintain accurate logging.
- **Fully Responsive:** Completely optimized for smooth usage across desktops, tablets, and mobile devices.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript, React.js, Tailwind CSS
- **Deployment & Hosting:** Vercel
